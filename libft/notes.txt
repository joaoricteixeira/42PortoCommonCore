ft_strlen: returns length of string
ft_strlcpy: copies string into buffer, always null-terminates
ft_strlcat: appends source string to the end of destination buffer, returns total length of string it tried to create
ft_strdup: allocates new copy of string
ft_strchr: finds first occurrence of char in string
ft_strrchr: finds last occurrence of char in string
ft_strncmp: compares n bytes of two strings
ft_atoi: converts string to int, handles +/-
ft_isalpha: checks if char is alphabetic
ft_isdigit: checks if char is digit
ft_isalnum: checks if char is alphanumeric
ft_isascii: checks if char is ASCII
ft_isprint: checks if char is printable
ft_toupper: converts char to uppercase if lowercase
ft_tolower: converts char to lowercase if uppercase
ft_memset: fills memory with a byte
ft_bzero: sets memory to zero
ft_memcpy: copies memory, undefined if overlapping
ft_memmove: copies memory, safe with overlapping
ft_memchr: finds first occurrence of byte in memory
ft_memcmp: compares memory blocks
ft_calloc: allocates and zeroes memory
ft_substr: returns substring from string
ft_strjoin: joins two strings into new string
ft_strtrim: trims characters from start/end of string
ft_split: splits string into array by delimiter
ft_itoa: converts int to string
ft_strmapi: applies function to each char of string, returns new string
ft_striteri: applies function to each char of string, in-place
ft_putchar_fd: writes char to file descriptor
ft_putstr_fd: writes string to file descriptor
ft_putendl_fd: writes string followed by newline to fd
ft_putnbr_fd: writes int as string to file descriptor
